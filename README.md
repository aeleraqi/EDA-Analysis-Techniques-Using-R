**Exploratory Data Analysis** techniques applied to the **Titanic dataset** using R.

## 📖 About

This project demonstrates comprehensive EDA on the famous Titanic dataset using R, covering data cleaning, visualization, and statistical analysis.

## 📂 Contents

- Data loading and initial inspection
- Missing value analysis and imputation
- Survival rate analysis by class, gender, and age
- Statistical tests for variable significance
- ggplot2 visualizations

## 🚀 Getting Started

```r
install.packages(c("ggplot2", "dplyr", "tidyr", "ggcorrplot"))
source("titanic_eda.R")
```

## 📊 Techniques Covered

| Technique | Description |

| Missing Value Analysis | Visualizing and imputing NA values |
| Distribution Plots | Histograms, boxplots, violin plots |
| Categorical Analysis | Bar charts, frequency tables |
